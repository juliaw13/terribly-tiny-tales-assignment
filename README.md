# terribly-tiny-tales-assignment
Required Module
1. Fetcher
2. Express
3. cors
4. body-parser
5. axios

Process
1. Open backend folder and run index1.js file using command "node index1.js".
2. Then open index.html file and enter roll number of the students(comma separated).
3. As you Enter roll no. of the students, a request been send to backend which will fetch the pass or fail details using testAPI and display the out on browser.
